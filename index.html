Atlas of Curiosities

This is a React app with a Rails backend to keep track of and add new interesting local history sites anywhere in the world utilizing the Google Maps API. The site also includes user authentication and authorization so that users can log in and record their own personal visits to the sites and view comments and ratings from others.

Star Trek: The Game

This is a pet project developed entirely using vanilla JavaScript on the HTML Canvas, no external libraries or frameworks, and minimal CSS. The game places you at the helm of the USS Enterprise as you fly around a fully orbiting solar system rescuing people from each planet while fending off attacks from an ever growing Klingon fleet.

Quotordle

This is a Wordle-like guessing game built in React and Rails where you are given a famous movie quote and have to guess the move it is from. This site also includes user authorization and authentication so that users can log their personal scores on the leaderboard.

GitHub Repo Demo

Kitchen Kingpin

This is a React app with a Rails backend to keep track of user made recipes, ingredients in the user's pantry, and a shopping list. The site also includes user authentication and authorization so that uers can log in and view/edit/add only their own recipes and view only their own shopping carts and pantries.
